Detailed Description

Clasă concretă care implementează interfața AbstractPainter.

View este responsabilă de afișarea grafică a obiectelor din joc, precum imaginile ASCII și textul. Aceasta poate fi extinsă sau adaptată pentru un motor grafic real.
